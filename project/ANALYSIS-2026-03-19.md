## 2. App Store Blockers

### B1. Turkish Language Strings (CRITICAL)
- AppCoordinator: lines 411, 419, 888, 937, 1227, 1231, 1511, 1675
- CommerceService: error descriptions (lines 128-140)
- **Fix:** Extract all strings to Localizable.strings, implement NSLocalizedString

### B2. Client-Side Trial Mechanism (CRITICAL)
- Trial stored as plaintext JSON: `~/Library/Application Support/ScreenMark/entitlements.json`
- No server validation, uses local `Date()` comparison
- Bypassable: delete file to restart, change system clock to extend
- **Fix:** Use StoreKit introductory offers or sign trial data with app-specific key

### B3. Missing Privacy Manifest (CRITICAL)
- No `PrivacyInfo.xcprivacy` file
- App uses: screen recording, input monitoring, microphone, file system
- **Fix:** Create PrivacyInfo.xcprivacy declaring all API usage

### B4. Missing Yearly Subscription (HIGH)
- `screenmark_pro_monthly` defined in code but not in `PurchaseCatalog.availableInV1`
- No yearly option defined at all
- **Fix:** Add `screenmark_pro_yearly` to catalog, include monthly+yearly+lifetime in V1

### B5. No Accessibility Support (HIGH)
- Zero VoiceOver labels, no high contrast mode, no reduced motion support
- **Fix:** Add accessibility labels/hints to all interactive elements

### B6. No Localization Infrastructure (HIGH)
- No Localizable.strings, no NSLocalizedString calls
- **Fix:** Create localization files, extract all user-facing text

### B7. Force Unwrap Crash (HIGH)
- AppCoordinator line 1573: `FileManager.default.urls(...).first!`
- **Fix:** Use `guard let` with fallback

### B8. Test Coverage ~4% (MEDIUM)
- 47 tests for 7,530 lines (9 source files have 0% coverage)
- **Fix:** Add tests for critical paths before release
