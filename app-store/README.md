# ScreenMark App Store Launch Files

This directory is the repo-owned source of truth for the first public Mac App Store release.

Included here:

- `product-matrix.json` — bundle ID, SKU, territories, pricing, intro-offer, and URL configuration
- `locales/*.json` — localized App Store metadata for the 8 launch locales
- `review-notes.md` — App Review explanation for permissions and privacy posture
- `submission-checklist.md` — account-side and release-side launch checklist
- `asset-manifest.json` — required screenshot and app preview deliverables

Public URLs used by the app and by App Store Connect:

- Marketing URL: `https://ceritmustafa.github.io/ScreenMark/`
- Support URL: `https://ceritmustafa.github.io/ScreenMark/support/`
- Privacy policy URL: `https://ceritmustafa.github.io/ScreenMark/privacy/`
